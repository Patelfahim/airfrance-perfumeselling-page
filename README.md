# ✈️ AirFrance Perfume Website UI

A modern and responsive perfume website inspired by AirFrance branding, focusing on clean UI and smooth user experience.

---

## 🚀 Features

- 🌐 Multi-language support (English / French)  
- 🎨 Modern gradient-based UI design  
- 🧭 Interactive navigation (History, Heritage, Products)  
- 📱 Fully responsive layout  
- ✨ Clean and minimal user interface  

---

## 🛠️ Tech Stack

HTML | CSS | JavaScript  

---

## 📌 Project Overview

This project was built to practice frontend development concepts such as UI design, responsiveness, and interactive elements. It simulates a real-world brand website experience.
